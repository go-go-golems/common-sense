# COMMON-SENSE STORES DATA, THIS IS COMMON SENSE
 
common-sense is a CMS framework. Using a YAML-based DSL, you can declaratively specify a CMS for a certain type of objects, which then exposes a REST API, web forms and a web application to browse that data.
